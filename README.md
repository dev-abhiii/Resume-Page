# Resume-Page (HTML only)
Chai Code Assignment

This is a single-page resume website built using only HTML.  
The layout includes sections such as About Me, Skills, Experience, Projects, Education, and Contact Details.

## How to View
1. Download or clone the repository
    - Copy the repositroy link (https://github.com/dev-abhiii/Resume-Page.git)
    - Open the folder in Terminal you want to copy the Repository in (cd/.../...)
    - run "Git clone https://github.com/dev-abhiii/Resume-Page.git"
2. Open the `index.html` file in any web browser
